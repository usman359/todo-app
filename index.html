My To-Do List

Stay on top of your day

    About This App

    This is a simple, beginner-friendly to-do app built with plain HTML, CSS, and JavaScript — no frameworks or build tools needed.

    Features

    • Add tasks with a single click or press Enter
    • Organize tasks by category — Work, Personal, or Study
    • Mark tasks as complete or delete them
    • Toggle between Light and Dark mode
    • Tasks saved automatically in your browser

    How It Works

    Tasks are saved in your browser using localStorage, so they stay after you refresh the page. The app uses plain JavaScript — no database or server required.

    Made as a learning project. Feel free to explore the code!
